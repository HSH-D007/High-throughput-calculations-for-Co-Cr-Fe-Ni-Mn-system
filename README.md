Data and code for "Machine learning-assisted high-throughput exploration of single-phase high-entropy alloys"

"CALPHAD results" includes the original dataset of predicted phases by CHALPHAD method for 1451 HEAs in the Co-Cr-Fe-Ni-Mn alloy system. 

"ML-Train" includes input data for training ML modeling.

"ML-Test" includes input data for testing ML modeling containing 72501 alloys.

"Predicted phase fractions by ML models" includes output of conducted ML models for phase prediction of 72501 alloys.

"Strength calculations" includes required dataset for calculating strength for single FCC phase alloys using the Labusch model.
